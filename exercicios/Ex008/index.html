PRINCIPAIS FORMATAÇÕES

Negrito

Para se colocar uma frase ou palavras
em negrito, basta utilizar a tag 'b'.
Veja o exemplo abaixo.
Hoje a noite eu vou usar apenas branco!

Outra opção para colocar coisas em negrito,
temos a tag 'strong', que é usada quando
tem algum significado, além de destacar palavras.

Itálico

Para se colocar uma frase ou palavras em
em itálico, basta utilizar a tag 'i'.
Veja o exemplo abaixo
Hoje eu vou à casa da minha Vó!

Outa opção para itálico tambem, é usar a
tag 'em', para dar ênfase, além de destacar.

Marcar Textos

Outra tag interessante é tag 'mark'. Ela é
uma espécie de marca texto para texto digitais,
veja o exemplo abaixo.
Uma vez FLAMENGO, sempre FLAMENGO

Textos Grandes e Pequenos

Para textos grandes e pequenos, podemos usar
a tag 'big' para textos grande, e a tag 'small' para textos menores.

Textos Deletados/Rasurados

Para textos rasurados ou deletados, usamos a tag
'del', veja o exemplo abaixo.
Hoje eu vou tomar um açaí.

Textos Inserido/Sublinhados

Para textos/palavras sublinhadas (o), vamos utilizar a
tag 'ins'. de forma semântica Veja abaixo
Olá, mundo!

E a tag 'u' para não semântica. Apenas forma, sem significado ou ênfase.
Olá, mundo!

Textos Sobrescritos

Vamos usar a tag 'sup' para sobrescrever/elevar. Veja abaixo
Quanto é x23?

Textos Subescritos

Para subescrever, vamos usar a tag 'sub'. Veja abaixo.
Eu quero H2O
