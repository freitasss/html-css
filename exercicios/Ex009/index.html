Trabalhando com listas

Listas Ordenadas

Hoje vamos aprender a criar lista ordenadas
Para criarmos, vamos abrir a tag de lista ordenada '' ol ''
e dentro dessa tag, vamos criar a tag de itens de lista
que é representada por ' li '. observe abaixo.

  1. abrir tag 'ol'
  2. abrir tag 'li' dentro da tag 'ol'
  3. e fazer a lista dentro da tag 'li'

Tambem é possível mudar o 'tipo de numeração' (Algarismos romanos maiúsculo,
minpusculo. letras maiúsculas e minúsculas, ).
Para isso, basta usar o paramentro 'type' dentro
da abertura da tag 'ol'.

  1. abrir tag 'ol'
  2. abrir tag 'li' dentro da tag 'ol'
  3. e fazer a lista dentro da tag 'li'

Outra observação é, caso queira que sua numeração começe com
um numero que não seja o numero 1, ainda dentro da tag 'ol' de abertura
use o parâmetro start, e entre aspas, coloque o numero que sua lista irá
começar. Veja abaixo.

  1. acordar
  2. escovar o dentes
  3. tomar café
  4. ir pro trabalho

Listas não ordenadas

As listas não ordenadas, são lista que não são 'numeradas'.
podemos usar simbolos para representar os itens. São eles: disc,
circle, square.

  • acordar
  • escovar o dentes
  • tomar café
  • ir pra escola

Os parâmetros para alterar o tipo de simbolos, o conceito é o mesmo,
bastas, dentro da tag 'ol' usar o paramentro 'type' e dentro das aspas
escrever os simbolos (disc, circle, square).
